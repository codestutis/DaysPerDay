# 🕒 DaysPerDay

**Ever wanted more than 24 hours in a day?**  
This web app calculates how many **pseudo-days** you can live in a single 25-hour day, based on your sleep and lifestyle habits. Optimize your time like a true time goblin (or stay human—your call).

---

## ✨ Features

### 🧠 User Inputs
- `Hours of sleep needed` — how much total sleep you require in 24 hours
- `Time it takes to fall asleep` — delay between bed and zzz
- `Time spent eating` — per pseudo-day
- `Time spent on hygiene` — per pseudo-day
- `Time spent on other stuff` — chores, transitions, etc.

---

### 📊 Outputs
- `Pseudo-days per 24-hour day`
- `Total hours slept per 24 hours`
- `Length of each pseudo-day` (sleep + awake + all delays)
- `Productivity rating` — how much more efficient you are:
  - Per real day
  - Per week
  - Per year
- `Daily schedule visualization` (timeline/pie chart)

---

### 🛠️ Tools & Integrations
- Export schedule to:
  - 📅 Google Calendar
  - 📱 Apple Calendar
- Compare different lifestyle modes:
  - 💤 Reality Mode (8 hours sleep, realistic hygiene)
  - 👹 Goblin Mode (minimum sleep, maximum grind)
  - 🚀 Productivity Mode (optimized cycles like polyphasic sleep)
